# Temporal Difference Learning

TD Learning for 2048 game

## Environment

- `g++ w/ std >= 11`

## Usage

- `make` for compile
- `./2048 -e {epochs, default = 100000} -s {state, before|after, default = before} -i {load weight path} -o {store weight path} -h {store history path}`
